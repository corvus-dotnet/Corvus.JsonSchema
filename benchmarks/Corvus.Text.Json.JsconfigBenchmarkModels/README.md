# Corvus.Text.Json.JsconfigBenchmarkModels

Strongly-typed C# types generated from the **jsconfig** JSON Schema
using the [Corvus.Text.Json](https://github.com/corvus-dotnet/Corvus.JsonSchema) code generator.

This package is published to support the [Sourcemeta JSON Schema benchmark suite](https://benchmark.sourcemeta.com/).
The types are pure generated code with no hand-written logic.

## Installation

```bash
dotnet add package Corvus.Text.Json.JsconfigBenchmarkModels
```

## Related Packages

| Package | Purpose |
|---|---|
| **Corvus.Text.Json** | Core library used by the generated types |
| **Corvus.Text.Json.SourceGenerator** | Source generator that produces these types |
| **Corvus.Json.CodeGenerator** | CLI tool for ahead-of-time code generation |
