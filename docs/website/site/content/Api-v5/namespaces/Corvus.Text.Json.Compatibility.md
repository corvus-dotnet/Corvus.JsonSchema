Backward-compatibility shims for migrating from Corvus.Json V4 (based on `System.Text.Json`). These types map V4's [`ValidationContext`](/api/v5/corvus-text-json-compatibility-validationcontext.html), [`ValidationResult`](/api/v5/corvus-text-json-compatibility-validationresult.html), and [`ValidationLevel`](/api/v5/corvus-text-json-compatibility-validationlevel.html) to V5 equivalents, allowing existing code to compile with minimal changes during migration.
