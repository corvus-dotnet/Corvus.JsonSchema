JsonLogic rule engine for evaluating business rules stored as JSON against data documents. Supports all standard JsonLogic operations with 100% conformance, plus custom operator extensions and both interpreted and code-generated evaluation modes.

Key types include [`JsonLogicEvaluator`](/api/v5/corvus-text-json-jsonlogic-jsonlogicevaluator.html) (the main entry point for evaluating rules) and [`JsonLogicRule`](/api/v5/corvus-text-json-jsonlogic-jsonlogicrule.html) (a wrapper for a parsed rule document).
