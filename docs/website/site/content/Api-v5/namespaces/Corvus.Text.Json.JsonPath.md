JSONPath (RFC 9535) query language evaluator and custom function extension API. Evaluate JSONPath expressions against JSON data with property access, wildcards, array slicing, recursive descent, filter expressions, and function extensions.

Key types include [`JsonPathEvaluator`](/api/v5/corvus-text-json-jsonpath-jsonpathevaluator.html) (the main entry point for evaluating expressions), [`JsonPathResult`](/api/v5/corvus-text-json-jsonpath-jsonpathresult.html) (a zero-allocation `ref struct` for collecting matched nodes), [`IJsonPathFunction`](/api/v5/corvus-text-json-jsonpath-ijsonpathfunction.html) (the interface for custom function extensions), [`JsonPathFunction`](/api/v5/corvus-text-json-jsonpath-jsonpathfunction.html) (delegate-based factory methods for creating custom functions), and [`JsonPathFunctionResult`](/api/v5/corvus-text-json-jsonpath-jsonpathfunctionresult.html) (the result type returned by custom functions).
