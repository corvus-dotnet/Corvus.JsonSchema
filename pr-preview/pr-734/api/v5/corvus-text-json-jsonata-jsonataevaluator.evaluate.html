Evaluate(string, JsonElement, int, int)

Evaluates a JSONata expression against JSON data.

public TElement Evaluate<TElement>(string expression, JsonElement data, int maxDepth, int timeLimitMs)
    where TElement : struct, IJsonElement<TElement>

Type Parameters

Parameter Description
TElement The type of the JSON element to return.

Parameters

Name Type Description
expression string The JSONata expression string.
data JsonElement The input JSON data.
maxDepth int The maximum recursion depth for function calls. Defaults to DefaultMaxDepth. Pass a lower value for safety, or a higher value for deeply recursive expressions. (optional)
timeLimitMs int The maximum time in milliseconds for expression evaluation. Zero means no limit. When the limit is exceeded, a JsonataException with code U1001 is thrown. (optional)

Returns

TElement

The result as a TElement. Returns a default value (with Undefined) if the expression produces no result.

Applies To

Evaluate(string, JsonElement, JsonWorkspace, IReadOnlyDictionary<string, JsonElement>, int, int)

Evaluates a JSONata expression against input data using the caller's workspace.

public TElement Evaluate<TElement>(string expression, JsonElement data, JsonWorkspace workspace, IReadOnlyDictionary<string, JsonElement> bindings, int maxDepth, int timeLimitMs)
    where TElement : struct, IJsonElement<TElement>

Type Parameters

Parameter Description
TElement The type of the JSON element to return.

Parameters

Name Type Description
expression string The JSONata expression string.
data JsonElement The input JSON data element.
workspace JsonWorkspace The workspace for intermediate document allocation. The returned element may reference documents owned by this workspace, so it remains valid only while the workspace is alive and has not been reset.
bindings IReadOnlyDictionary<string, JsonElement> Optional pre-defined variable bindings (name → value). (optional)
maxDepth int Maximum recursion depth (default 500). (optional)
timeLimitMs int Maximum evaluation time in milliseconds (0 = no limit). (optional)

Returns

TElement

The evaluation result as a TElement, or default if the result is undefined.

Remarks

Because the result is not cloned, this overload avoids allocation at the evaluation boundary. The caller is responsible for ensuring the workspace outlives any use of the returned element.

Evaluate(byte[], JsonElement, JsonWorkspace, string, int, int)

Evaluates a JSONata expression provided as pre-encoded UTF-8 bytes. This overload bypasses the string-to-UTF-8 transcode. When a cacheKey is provided, the compiled expression is cached under that key (same cache as the string overloads). When cacheKey is null, the expression is compiled fresh each invocation — ideal for cold-start scenarios where each expression is evaluated once.

public TElement Evaluate<TElement>(byte[] utf8Expression, JsonElement data, JsonWorkspace workspace, string cacheKey, int maxDepth, int timeLimitMs)
    where TElement : struct, IJsonElement<TElement>

Type Parameters

Parameter Description
TElement The type of the JSON element to return.

Parameters

Name Type Description
utf8Expression byte[] The JSONata expression as UTF-8 bytes.
data JsonElement The input JSON data element.
workspace JsonWorkspace The workspace for intermediate document allocation. The returned element may reference documents owned by this workspace, so it remains valid only while the workspace is alive and has not been reset.
cacheKey string Optional cache key. When non-null, the compiled expression is stored (and retrieved on subsequent calls) using this key. The caller is responsible for ensuring the key uniquely identifies the expression. (optional)
maxDepth int Maximum recursion depth (default 500). (optional)
timeLimitMs int Maximum evaluation time in milliseconds (0 = no limit). (optional)

Returns

TElement

The evaluation result as a TElement, or default if the result is undefined.
