JMESPathCodeGenHelpers

Definition

Namespace: Corvus.Text.Json.JMESPath
Assembly: Corvus.Text.Json.dll

Public helper methods used by JMESPath source-generated evaluation code.

public static class JMESPathCodeGenHelpers

Remarks

These methods are called from code emitted by JMESPathCodeGenerator and JMESPathSourceGenerator. They provide the same semantics as the runtime Compiler — both share the same implementations to guarantee identical behaviour. This class is not intended for direct use by application code.
