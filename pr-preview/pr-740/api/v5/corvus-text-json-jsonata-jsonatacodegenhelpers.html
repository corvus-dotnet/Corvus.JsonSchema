JsonataCodeGenHelpers

Definition

Namespace: Corvus.Text.Json.Jsonata
Assembly: Corvus.Text.Json.dll

Public helper methods for code-generated JSONata evaluators.

public static class JsonataCodeGenHelpers

Remarks

This class provides the runtime operations that generated JSONata evaluation code calls. It bridges between the public JsonElement API and the internal JSONata evaluation machinery. Generated code emitted by JsonataCodeGenerator calls these helpers for operations that are too complex to inline (path navigation with auto-flattening, arithmetic with type checking, string coercion, etc.). Simple operations like literal creation and property access are emitted inline.
