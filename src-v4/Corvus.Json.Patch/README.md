# Corvus.Json.Patch

> **Note:** This is a V4 package. For new projects, consider using the V5 equivalent from the Corvus.Text.Json family.

Support for the Json Patch Specification

## Installation

```bash
dotnet add package Corvus.Json.Patch
```
