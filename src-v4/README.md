This folder contains the source code for the Corvus.JsonSchema components that were available in V4 and earlier, including what is colloquially known as the V4 code generation engine ("Corvus.JsonSchema").

This include the core infrastructure for code generation, the V4 engine (Corvus.Json.CodeGeneration.CSharp) and related types.

The command line tool Corvus.Json.CodeGenerator is now shipped from the ../src/Corvus.Json.CodeGenerator folder, and allows you to configure either the V4 or the V5 engine.
