# Corvus.Text.Json.Compatibility

Compatibility layer for Corvus.Text.Json that provides bridge types and helpers for interoperating with `System.Text.Json` types and APIs.

## Installation

```bash
dotnet add package Corvus.Text.Json.Compatibility
```

## Target Frameworks

- .NET 10.0, 9.0, 8.0
- .NET Standard 2.0

## Related Packages

| Package | Purpose |
|---|---|
| **Corvus.Text.Json** | Core runtime library |
