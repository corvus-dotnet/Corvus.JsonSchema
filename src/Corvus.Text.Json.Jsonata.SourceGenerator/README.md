# Corvus.Text.Json.Jsonata.SourceGenerator

Roslyn incremental source generator that produces optimized C# evaluation code from JSONata expression files at build time.

## Installation

```bash
dotnet add package Corvus.Text.Json.Jsonata.SourceGenerator
```

## Related Packages

| Package | Purpose |
|---|---|
| **Corvus.Text.Json.Jsonata** | JSONata query and transformation language |
| **Corvus.Text.Json.Jsonata.CodeGeneration** | Code generation engine for JSONata |
