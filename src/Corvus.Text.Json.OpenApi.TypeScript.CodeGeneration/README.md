# Corvus.Text.Json.OpenApi.TypeScript.CodeGeneration

The TypeScript OpenAPI client emitter. It is the TypeScript counterpart of the C# emitters in the version packages, and it plugs into the same generator core. `TypeScriptApiEmitter` implements `IClientEmitter` and is driven by the shared `OpenApiWalkerBase` walk and `ClientEmitDriver`, so a TypeScript client is generated from the identical operation IR as the C# client and composes byte-for-byte wire-compatible requests.

## Key types

- **`TypeScriptApiEmitter`** implements `IClientEmitter`. It emits idiomatic TypeScript: a per-operation request module, a response type with an exhaustive `match`, a per-tag client class over an injected transport, a server-URI factory, response links, and streaming bodies as `AsyncIterable`.
- **`TypeScriptSchemaTypeResolver`** implements `ISchemaTypeResolver`. It maps a schema pointer to the generated TypeScript type name and its module.
- **`TypeScriptApiEmitterOptions`** carries the emit options, including the client-runtime module specifier and per-type module splitting.

The request and response types are generated by the TypeScript model engine (`Corvus.Text.Json.TypeScript.CodeGeneration`), and the generated client imports its byte-native transport from the `@endjin/corvus-json-client-runtime` npm package.

## Related packages

- `Corvus.Text.Json.OpenApi.CodeGeneration` is the shared generator core this emitter plugs into.
- `Corvus.Text.Json.TypeScript.CodeGeneration` is the JSON Schema to TypeScript model engine used for the request and response types.
- `Corvus.Text.Json.OpenApi30`, `.OpenApi31`, and `.OpenApi32` host the equivalent C# emitters.
