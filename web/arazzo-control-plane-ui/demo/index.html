Arazzo Control Plane

UI kit demo — running against an in-memory mock API (no server)
Try the faulted run’s Resume…, cancel a running one, or Purge… old completed runs.

The kit talks to the mock through the client’s fetch hook — in a real app you’d set panel.authProvider = () => 'Bearer …' instead and point base-url at your control-plane host. See ../README.md.
